Resolvendo o Problema de Execução do Streamlit

Problema: Antigamente você usava streamlit run arquivo.py, mas agora precisa usar python -m streamlit run arquivo.py.
Solução: Isso geralmente acontece quando o executável do Streamlit não está no PATH do sistema. Aqui estão várias soluções possíveis.

Soluções Possíveis

1 Verificar a Instalação

Primeiro, verifique se o Streamlit está instalado corretamente:

pip install streamlit

Ou se você estiver usando conda:

conda install -c conda-forge streamlit

2 Verificar se o Streamlit está Acessível

Teste se o Streamlit responde com:

python -m streamlit --version

Se isso funcionar, o Streamlit está instalado mas não está no PATH.

3 Adicionar ao PATH do Sistema

Windows
Linux/Mac

No Windows, adicione o diretório de scripts do Python ao PATH:

  1. Pressione Win + R, digite sysdm.cpl e pressione Enter
  2. Clique em "Variáveis de Ambiente"
  3. Na seção "Variáveis do sistema", encontre a variável PATH e clique em "Editar"
  4. Adicione o caminho para a pasta Scripts do Python (ex: C:\Python39\Scripts)
  5. Reinicie o prompt de comando

No Linux ou Mac, adicione o diretório de scripts ao PATH:

  1. Abra o arquivo ~/.bashrc ou ~/.zshrc em um editor de texto
  2. Adicione a linha: export PATH="$PATH:/caminho/para/python/scripts"
  3. Salve o arquivo e execute: source ~/.bashrc (ou source ~/.zshrc)

4 Criar um Arquivo Batch/Shell Script

Windows (.bat)
Linux/Mac (.sh)

Crie um arquivo streamlit.bat com o seguinte conteúdo:

@echo off
python -m streamlit %*

Salve este arquivo em um diretório que esteja no seu PATH.

Crie um arquivo streamlit.sh com o seguinte conteúdo:

#!/bin/bash
python -m streamlit "$@"

Torne-o executável e coloque em um diretório no PATH:

chmod +x streamlit.sh
sudo mv streamlit.sh /usr/local/bin/streamlit

5 Usar um Alias

No Linux ou Mac, você pode criar um alias no seu arquivo de shell:

echo 'alias streamlit="python -m streamlit"' >> ~/.bashrc
source ~/.bashrc

Ou para zsh:

echo 'alias streamlit="python -m streamlit"' >> ~/.zshrc
source ~/.zshrc

6 Script Python Auxiliar

Crie um script Python que execute o Streamlit para você:

#!/usr/bin/env python3
import sys
import subprocess

def main():
    if len(sys.argv) < 2:
        print("Uso: python run_streamlit.py <arquivo>")
        sys.exit(1)
    
    file_path = sys.argv[1]
    subprocess.run([sys.executable, "-m", "streamlit", "run", file_path])

if __name__ == "__main__":
    main()

Salve como run_streamlit.py e use com:

python run_streamlit.py seu_script.py

7 Verificar Ambiente Virtual

Se você estiver usando um ambiente virtual, certifique-se de que está ativado:

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate

Depois ative o ambiente virtual, reinstale o Streamlit:

pip install streamlit

Testando a Solução

Após aplicar uma das soluções, teste se o comando funciona:

streamlit --version

Se retornar a versão do Streamlit, o problema foi resolvido.

Dica: A solução mais comum é adicionar o diretório de scripts do Python ao PATH do sistema (Solução 3).
